# CSCA小站网站项目

## 项目简介
CSCA小站是一个为来华留学本科入学学业水平测试（CSCA）提供学习资源和官方资讯的网站。网站提供考试模拟、练习中心、官方资讯等功能，支持中英文双语切换。

## 最新更新
- **2025-11-10**：为数学科目新增来自 `test_math/math_1.json` 的“第三套”模拟题，涵盖集合、三角函数、解析几何等核心考点，并补充解析说明。

cscauni/

├── styles.css          # 网站样式文件

## 功能说明

### 1. 固定导航栏（A区）
- **位置**：页面顶部固定悬浮
- **功能**：
  - Logo展示
  - 导航菜单：首页、考试模拟、官方资讯
  - 登录/注册按钮（右上角）
  - 语言切换按钮（中文/English）
  - 登录后显示头像和个人中心

### 2. 首屏Banner区（B区）
- **功能**：
  - 展示CSCA考试主标题和副标题
  - 考试倒计时（动态更新）
  - 立即报名按钮
  - 官方公告滚动栏（3条最新资讯轮播）

### 3. 科目学习区（C区）
- **布局**：2行2列卡片式，共4门主科
- **功能**：
  - 每门学科卡片展示
  - 鼠标悬停时卡片浮起并显示二级菜单
  - 点击"进入练习"按钮跳转到模拟练习页面
  - 点击"模拟题库"菜单项跳转到模拟练习页面
  - 点击"考试说明"菜单项显示提示信息

### 4. 官方资讯区（E区）
- **布局**：三栏新闻流
  - 左栏：教育部/留基委公告
  - 中栏：高校新闻
  - 右栏：使领馆动态与签证信息
- **功能**：点击文章标题跳转到详情页

### 5. 页脚（G区）
- **内容**：
  - Logo和简介
  - 联系方式（邮箱、热线）
  - 快速导航链接
  - 语言切换
  - 社交媒体链接
  - 版权信息

### 6. 模拟练习页面（practice.html）
- **功能**：
  - 科目选择：支持中文、数学、物理、化学四个科目，可通过按钮或URL参数切换
  - 套题选择：每个科目提供多套试题（如第一套、第二套、第三套等），可快速切换不同练习套题
  - 题目展示：支持单选题、多选题两种客观题型，题干与选项均动态渲染
  - 答题功能：
    - 选择题支持单选和多选
    - 可以标记题目以便后续查看
    - 可以清除当前题目答案
  - 计时功能：
    - 默认60分钟倒计时
    - 剩余时间少于10分钟时显示警告样式
    - 时间到自动提交试卷
    - 支持本地存储保存剩余时间
  - 题目导航：
    - 题目编号网格，快速跳转到任意题目
    - 显示当前题目、已答题、已标记题目的状态
    - 上一题/下一题按钮
  - 进度统计：
    - 实时显示已答题数、未答题数、已标记数
    - 进度条可视化显示答题进度
  - 提交功能：
    - 提交前显示确认弹窗，展示答题统计
    - 自动计算分数和正确率
    - 提交后禁用所有答题功能
  - 查看答案：
    - 显示所有题目的正确答案和用户答案
    - 标注答案正确或错误
    - 显示题目解析
  - 其他功能：
    - 重新开始：清除所有答题记录并重置
    - 返回首页：确认后返回首页
    - 自动保存：页面关闭时自动保存答题记录和剩余时间

## 使用方法

### 本地运行
1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 或者使用本地服务器运行（推荐）：
   ```bash
   # 使用Python启动本地服务器
   python -m http.server 8000
   # 然后在浏览器访问 http://localhost:8000
   ```

### 文件说明
- **index.html**：主页面HTML结构，包含所有区域的内容
- **practice.html**：模拟练习页面HTML结构，包含题目展示、答题、计时等功能
- **news-detail.html**：新闻详情页面HTML结构
- **styles.css**：首页样式文件，定义颜色、布局、动画效果
- **practice.css**：模拟练习页面样式文件，定义练习页面的布局和样式
- **script.js**：首页JavaScript交互功能，包括倒计时、轮播、悬停效果、页面跳转等
- **practice.js**：模拟练习页面JavaScript交互功能，包括题目管理、答题、计时、提交等

## 模拟练习页面使用说明

### 进入练习页面
1. 在首页的科目学习区，点击任意科目的"进入练习"按钮
2. 或点击科目卡片悬停时显示的"模拟题库"菜单项
3. 页面会自动跳转到模拟练习页面，并传递科目名称参数

### 答题操作
1. **选择题**：直接点击选项即可选择答案，单选题只能选择一个，多选题可以选择多个
2. **填空题**：在文本输入框中输入答案
3. **标记题目**：点击"标记题目"按钮可以标记当前题目，方便后续查看
4. **清除答案**：点击"清除答案"按钮可以清除当前题目的答案

### 题目导航
1. **题目编号网格**：右侧边栏显示所有题目的编号，点击可以快速跳转
   - 绿色：已答题
   - 黄色：已标记
   - 蓝色：当前题目
   - 白色：未答题
2. **上一题/下一题**：点击导航栏的按钮可以切换题目

### 提交试卷
1. 点击右上角的"提交试卷"按钮
2. 在确认弹窗中查看答题统计
3. 确认提交后，系统会自动计算分数并显示结果
4. 提交后无法再修改答案

### 查看答案
1. 点击右侧边栏的"查看答案"按钮
2. 在弹窗中查看所有题目的正确答案和您的答案
3. 系统会标注答案是否正确，并显示题目解析

### 其他功能
- **重新开始**：清除所有答题记录，重新开始练习
- **返回首页**：返回网站首页
- **自动保存**：系统会自动保存您的答题记录和剩余时间，即使关闭页面也不会丢失
